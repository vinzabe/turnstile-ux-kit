# Vanilla JS Example

A simple vanilla JavaScript example demonstrating the Turnstile UX Kit.

## Running

```bash
cd examples/vanilla
# Start a simple HTTP server
python -m http.server 8080
# or
npx serve .
```

Open http://localhost:8080 in your browser.

## Features Demonstrated

- Inline widget rendering
- Token handling
- Theme switching
- Language switching
- Telemetry events
- Error handling

## Files

- `index.html` - Main demo page

## Notes

Uses a test Turnstile site key that accepts any valid challenge.
