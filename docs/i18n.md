# Localization Guide

## Adding a New Locale

1. Create a JSON file in `src/i18n/locales/`:
   ```bash
   src/i18n/locales/fr.json
   ```

2. Add translations following the structure:
   ```json
   {
     "turnstile": {
       "heading": "Vérifiez que vous êtes humain",
       "description": "Veuillez compléter la vérification ci-dessous"
     },
     "common": {
       "loading": "Chargement...",
       "cancel": "Annuler"
     }
   }
   ```

3. Load and use:
   ```javascript
   const i18n = new I18n();
   await i18n.loadLocale('fr', await fetch('./i18n/fr.json').then(r => r.json()));
   i18n.setLocale('fr');
   ```

## Variable Interpolation

Use `{{variable}}` syntax:

```json
{
  "message": "Please wait {{seconds}} seconds"
}
```

Usage:
```javascript
i18n.t('message', { seconds: 30 }); // "Please wait 30 seconds"
```

## RTL Support

The kit automatically handles RTL languages. Set the `dir` attribute:

```html
<html lang="ar" dir="rtl">
```

CSS automatically adjusts for RTL.

## Supported Locales

Currently supported:
- English (`en`)
- Spanish (`es`)

Contributions for additional locales welcome!
