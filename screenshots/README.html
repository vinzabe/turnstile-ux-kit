How to Take Screenshots

Follow these steps to capture screenshots of the Turnstile UX Kit:

Step 1: Open Demo

Open the demo.html file in your web browser by double-clicking it or using the File menu.

Step 2: Capture Screenshots

For each section of the demo page:

  • Position the section in the center of your viewport
  • Use your browser's screenshot tool:
    • Windows: Win + Shift + S or Print Screen
    • Mac: Cmd + Shift + 4
    • Linux: Print Screen
  • Save each screenshot with a descriptive name in the screenshots/ folder

Recommended Screenshots

  2. Dark Theme: The second section (scroll down) with dark background
  3. Rate Limited: The yellow warning section
  4. WAF Blocked: The red security section
